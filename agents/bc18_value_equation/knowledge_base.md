# BC18 Knowledge Base: Value Equation Optimizer

> Framework encoder: Hormozi Value Equation 4 lever + Brunson Epiphany Bridge.

## 1. Hormozi Value Equation

```
                Dream Outcome × Perceived Likelihood
Value = ───────────────────────────────────────────
                 Time Delay × Effort & Sacrifice
```

Numerator UP + Denominator DOWN = Value perceived ∞.

## 2. 4 Levers Audit

### Lever 1: Dream Outcome (numerator, MAXIMIZE)

**Score 1-10**:
- 10: cụ thể + measurable + có timeline + relate persona
- 7-8: cụ thể + measurable nhưng vague timeline
- 4-6: general benefit + vague
- 1-3: generic "thành công"

**Improve actions**:
- Add specific number (15tr/tháng, 100 đơn/tháng)
- Add timeline (90 ngày, 6 tháng)
- Add scenario detail ("đứng pitch với khách Mỹ tại sự kiện")
- Use persona language verbatim

### Lever 2: Perceived Likelihood (numerator, MAXIMIZE)

**Score 1-10**:
- 10: 10+ testimonial specific + case study same persona + money-back guarantee + track record founder
- 7-8: 5+ testimonial + 1-2 case study
- 4-6: vague testimonial + no case study
- 1-3: "trust me"

**Improve actions**:
- Add testimonial specific (tên, số, thời gian)
- Add case study same persona
- Add guarantee conditional/unconditional
- Show founder track record (Anna: 33K Speakout + 5 venture + CAMAS 27 agent)
- Show step-by-step roadmap

### Lever 3: Time Delay (denominator, MINIMIZE)

**Score 1-10**:
- 10: instant gratification 24-48h + milestone clear week 1/4/12
- 7-8: quick win 1 tuần + roadmap rõ
- 4-6: vague timeline
- 1-3: no time guarantee

**Improve actions**:
- Add Day 1 deliverable (vd: Day 1 K2 = có 1 sản phẩm draft)
- Add Week 1 milestone
- Add 30-day quick win benchmark
- Add Done-with-you elements

### Lever 4: Effort & Sacrifice (denominator, MINIMIZE)

**Score 1-10**:
- 10: minimal effort + done-for-you + 5-10 phút/ngày + automate via AI
- 7-8: template sẵn + accountability + low daily commit
- 4-6: cần học nhưng có support
- 1-3: 100% self-effort

**Improve actions**:
- Add done-for-you templates (Canva, Notion, Figma)
- Add weekly accountability call
- Add AI automation (Anna's edge với AIOS)
- Reduce daily time commitment (5-10 phút quy luật)
- Replay record nếu miss buổi
- Pre-built workflow

## 3. Epiphany Bridge (Brunson)

Bridge từ skeptical → believer trong 5 steps:

1. **Backstory**: Where customer is now (pain state)
2. **Wall**: Cái cản trở họ
3. **Epiphany**: Aha moment họ cần
4. **Plan**: How offer helps
5. **Conflict + Achievement**: Struggle + Win

Apply Epiphany Bridge để increase Likelihood:
- Show customer hành trình tương tự
- Wall = same pain họ đang có
- Epiphany = framework Anna teach
- Plan = roadmap clear
- Achievement = testimonial result

## 4. Value Equation Audit Workflow

Step 1: Read existing offer + current Dream Outcome statement
Step 2: Score 4 levers (1-10 mỗi cái)
Step 3: Identify weakest lever (lowest score)
Step 4: Recommend 3-5 specific actions để improve weakest
Step 5: Calculate projected value lift (new_value / old_value)
Step 6: Output revised offer statement

## 5. Common pitfalls

❌ KHÔNG nên:
- Score 10/10 mọi lever (= bias, audit không có giá trị)
- Improve toàn bộ levers (= scattered)
- Recommend vague action ("làm tốt hơn")
- Ignore persona context khi audit

✅ Nên:
- Honest score per lever
- Focus weakest lever first (biggest ROI)
- Recommend concrete action với example
- Cite persona pain/joy khi audit
- Compare with Anna's known offer patterns
